# Unsupervised-Learning-using-Clustering-and-Dimensionality-Reduction
## Project Overview

This project applies unsupervised machine learning techniques to bank customer data to discover hidden patterns and customer segments. Principal Component Analysis (PCA) is used for dimensionality reduction, while K-Means clustering groups customers based on similar characteristics.

The project demonstrates how dimensionality reduction can simplify high-dimensional data and improve cluster visualization and interpretation.

---

## Objectives

- Explore and preprocess bank customer data.

- Apply Principal Component Analysis (PCA) for dimensionality reduction.

- Analyze principal components for feature representation.

- Implement K-Means clustering for customer segmentation.

- Determine the optimal number of clusters using the Elbow Method and Silhouette Analysis.

- Visualize clusters and principal component projections.

---

## Techniques Used

### Principal Component Analysis (PCA)

- Standardized the dataset.

- Computed principal components to reduce dimensionality.

- Used principal component projections for visualization and clustering.

### K-Means Clustering

- Applied K-Means to identify customer groups.

- Evaluated different cluster sizes.

- Selected the optimal number of clusters using:

  - Elbow Method

  - Silhouette Score
